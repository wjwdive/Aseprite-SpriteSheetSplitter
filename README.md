# Aseprite Sprite Sheet Splitter

一个强大且精确到像素的 Aseprite 插件，用于将精灵图表（Sprite Sheet）切分为独立的元素。支持实时预览、4 向外边距设置以及多种导出模式。

## ✨ 插件特色

- **实时可视化预览**：在调整参数时，画布上会实时显示切割线，确保像素级精确，防止裁切错误。
- **灵活的外边距与间距**：支持 Top（上）、Leading（左）、Bottom（下）、Trailing（右）四方向外边距设置，以及自定义行列间距（Gap X/Y）。
- **非阻塞式交互**：窗口开启时仍可自由缩放、平移画布，方便检查每一个像素细节。
- **多种导出模式**：
  - **导出为独立文件**：将每个切片保存为单独的 PNG 文件。
  - **合并为新精灵图表**：将切片按指定的 M 行 N 列重新排列并生成新文件。
  - **导出为动画帧**：将每个切片自动转换为新文件中的动画帧序列。
- **自动裁剪 (Autocrop)**：支持自动去除切片周围的透明像素。
- **自定义预览色**：提供多种高对比度预设颜色及自定义色盘，确保在任何底色上都能清晰看到切割线。

## 🚀 适用场景

- **游戏素材拆分**：将整张精灵图表拆分为独立的单位素材。
- **动画序列整理**：将水平或垂直排列的动画帧转换为 Aseprite 动画帧序列。
- **UI 组件提取**：从 UI 图集中精确提取带有间距和边距的组件。
- **素材重排**：改变素材的行列布局以适应不同的引擎要求。

## 📋 环境要求

- **Aseprite 版本**：推荐 v1.3.17 或更高版本。
- **操作系统**：Windows, macOS, Linux (只要能运行 Aseprite 即可)。

## 🛠️ 使用说明

1. **安装插件**：
   - 将 `SpriteSheetSplitter.lua` 文件复制到 Aseprite 的脚本目录：`文件 (File) -> 脚本 (Scripts) -> 打开脚本文件夹 (Open Scripts Folder)`。
   - 点击 `文件 (File) -> 脚本 (Scripts) -> 刷新脚本 (Rescan Scripts Folder)`。
2. **运行插件**：
   - 打开需要切分的精灵图。
   - 在菜单中选择 `文件 (File) -> 脚本 (Scripts) -> SpriteSheetSplitter`。
3. **参数配置**：
   - **Rows/Cols**: 输入原始图表的行数和列数。
   - **Outer Margins**: 设置图像四周的像素外边距。
   - **Inner Spacing**: 设置元素之间的像素间距。
   - **Output Settings**: 选择导出模式（独立文件、合并图表或动画帧）。
4. **执行**：点击 `Split and Export` 即可完成。
